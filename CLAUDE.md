# 项目协作约定

## 原型文件同步规则

本项目有两个并行的 HTML 原型文件：
- `index.html` — 主原型（用于日常演示和评审）
- `prototype.html` — 备用原型

**凡修改任一原型中的内容，必须同步到另一个文件。** 两者结构基本一致，以"需求说明"（`prdTab`）区域为例：tab nav 和对应的 `v-if` 内容块两处都要改。只改一个视为未完成。

## UI / 文档风格

- **不使用 emoji 图标**：所有原型（index.html、prototype.html）、PRD 文档、提示文案等，禁止出现 emoji 图标（如 📋 🔍 💡 ⚠️ 📌 ✅ ❌ 等）。
- **关闭按钮的 ✕ 与勾选符号 ✓ 例外**：这两个为功能性符号，可以保留。
- **如确需图标**：使用 [lucide icons](https://lucide.dev/) 替代 emoji。
